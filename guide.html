Guide

装饰、发文和个性化设置

下面是你后续维护博客最常用的步骤。这个静态模板可以直接改 HTML；如果换成 Hexo，则主要改配置文件、主题文件和 Markdown 文章。

网页装饰

  1. 替换 assets/hero-workspace.png，就能更换首页大图。
  2. 修改 assets/styles.css 里的 --accent、--paper、--ink 等变量，可以快速换色。
  3. 在 index.html 中调整站名、简介、按钮文案和首页模块。
  4. 想加头像，可在 assets 放入图片，并在首页或侧栏添加 <img>。

发布文章

  1. 静态模板：复制 article-github-pages-hexo.html，改成新文件名。
  2. 在新文章里修改标题、日期、标签和正文。
  3. 到 posts.html 复制一张文章卡片，链接到新文章。
  4. Hexo 方式：运行 hexo new "文章标题"，然后编辑 source/_posts 里的 Markdown 文件。

个性化设置

  1. 站点名称：搜索并替换“星屿札记”。
  2. 导航菜单：修改每个页面顶部 nav 中的链接。
  3. 页脚信息：修改每个页面底部 footer。
  4. 明暗主题和强调色：逻辑在 assets/site.js，样式在 assets/styles.css。

部署到 GitHub Pages

  1. 新建仓库，例如 username.github.io。
  2. 把本网站文件上传到仓库根目录。
  3. 在 GitHub 仓库中进入 Settings → Pages。
  4. 选择部署分支，保存后等待 Pages 构建完成。
